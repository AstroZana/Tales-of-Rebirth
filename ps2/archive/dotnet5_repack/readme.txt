Compile and run tor.exe in the same folder containing DAT folder with the extracted DAT.BIN from Tales of Rebirth (PS2).
To run this console application, you'll need the following files in the same folder as well:
1. dat.json
2. new_SLPS_254.50
3. TOR.exe
4. TOR.dll
5. TOR.deps.json
6. TOR.runtimeconfig.json

After compiling, you can run this on Windows, Linux, MacOS since this is .NET5
